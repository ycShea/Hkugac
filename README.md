# My Portal

🔗 **[Click here to access the Live Portal](https://ycShea.github.io/Hkugac/)**

A lightweight, web-based dashboard designed to provide quick access to essential resources for the ADC, Math, IT Admin, and PC teams. 

## 🚀 Features
- **Interactive Interface:** Collapsible sections with toggle triangles to keep the view clean.
- **Centralized Links:** Direct access to Google Drive folders and Team Spreadsheets.
- **Responsive Design:** Built with Bootstrap 5 to ensure it works on desktops, tablets, and mobile devices.
- **Clean UI:** Organized sections with color-coded buttons for easy navigation.

## 🛠️ Tech Stack
- **HTML5**
- **CSS3** (Custom styling + Bootstrap 5)
- **JavaScript** (Bootstrap Bundle for toggle functionality)
- **Hosted via:** GitHub Pages

## 📖 How to Use
1. **School Admin:** Access core administrative documents and school-wide management tools.
2. **ADC:** Click the header to reveal the link to the primary Google Drive folder.
3. **Math:** Access S6 Student Data Analysis and other math-related resources.
4. **IT Admin:** Direct link to the Student Portfolio spreadsheet.
5. **PC Team:** Direct link to the Cell Phone Lock Up tracking sheet.
6. **Footer:** Quick access to the "Links of documents 2025-26" master list.

## 🔧 Deployment
To host this yourself:
1. Upload `index.html` to your GitHub repository.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, set the source to `Deploy from a branch` and select `main`.
4. Your site will be live at `https://ycShea.github.io/Hkugac/`.
